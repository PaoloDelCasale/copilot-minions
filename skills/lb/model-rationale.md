# Low-budget model rationale

This profile is inspired by the economical role routing in
`nsEytgXm/subagents_configs`:

- Luna low for mechanical work.
- Luna medium for read-heavy exploration.
- Luna high for implementation.
- Sol low for independent review.

The existing minions flow also needs architecture and planning roles; both use Luna
high rather than adding a costlier model. The frontier remains Sol medium because it
owns decomposition and user decisions. Unlike the source configuration, this profile
does not split validation into a separate agent or make review selective.
